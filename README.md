## Todo-List Project

#### A clean, interactive, and user-friendly task management web application created to help you organize your daily activities. This project is built using standard web technologies and runs entirely in your browser.

## Features

#### Add Tasks: Easily type and add new items to your daily to-do list.

#### Mark as Complete: Visually strike through or check off tasks as you finish them.

#### Delete Tasks: Remove items from your list once they are no longer needed.

#### Lightweight: Fast loading and runs instantly without needing a server.

## Technologies Used

#### HTML5 (index.html): Structures the web page and the main task input interface.  

#### CSS3 (style.css): Provides the layout, animations, colors, and visual styling.  

#### JavaScript (script.js): Contains the dynamic logic for adding, removing, and updating tasks.
